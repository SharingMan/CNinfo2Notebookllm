## 🚀 手动上传步骤

### 第一步：创建 NotebookLM 笔记本

1. 打开浏览器，访问：https://notebooklm.google.com/
2. 点击 **"新建笔记本"** 或 **"New Notebook"** 按钮
3. 输入笔记本名称：**世纪华通 财务报告**

### 第二步：上传 PDF 文件

1. 在笔记本页面，点击 **"添加来源"** 或 **"Add Source"**
2. 选择 **"上传文件"** 或 **"Upload files"**
3. 选择本文件夹中的所有 7 个 PDF 文件
4. 等待上传完成（可能需要几分钟）

### 第三步：配置 AI 财务分析师（可选但推荐）

1. 打开 `financial_analyst_prompt.txt` 文件
2. 复制其中的所有内容
3. 在 NotebookLM 中，点击设置或配置按钮
4. 找到 **"自定义指令"** 或 **"Custom Instructions"** 选项
5. 粘贴提示词内容
6. 保存设置
