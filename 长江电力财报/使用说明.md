## 🚀 手动上传步骤

### 第一步：创建 NotebookLM 笔记本

1. 打开浏览器，访问：https://notebooklm.google.com/
2. 点击 **"新建笔记本"** 或 **"New Notebook"** 按钮
3. 输入笔记本名称：**长江电力 财务报告**

### 第二步：上传 PDF 文件

1. 在笔记本页面，点击 **"添加来源"** 或 **"Add Source"**
2. 选择 **"上传文件"** 或 **"Upload files"**
3. 选择本文件夹中的所有 7 个 PDF 文件
4. 等待上传完成（可能需要几分钟）

### 第三步：配置 AI 财务分析师（可选但推荐）

1. 打开 `financial_analyst_prompt.txt` 文件
2. 复制其中的所有内容
3. 在 NotebookLM 中，点击设置或配置按钮
4. 找到 **"自定义指令"** 或 **"Custom Instructions"** 选项
5. 粘贴提示词内容
6. 保存设置

## 💡 长江电力分析重点

长江电力是中国最大的水电上市公司，分析时可以重点关注：

### 🔍 核心指标

1. **发电量与上网电量**
   - "长江电力的发电量趋势如何？"
   - "来水情况对业绩的影响有多大？"

2. **现金分红**
   - "长江电力的分红政策如何？"
   - "股息率是否具有吸引力？"

3. **资产质量**
   - "三峡大坝等核心资产的价值如何？"
   - "资产负债率是否健康？"

4. **盈利稳定性**
   - "公司的盈利能力是否稳定？"
   - "ROE（净资产收益率）表现如何？"

### 📊 行业特点

- **公用事业属性**：收入相对稳定，受经济周期影响较小
- **来水影响**：发电量受长江流域降水量影响
- **政策支持**：清洁能源，符合国家"双碳"战略
- **高分红**：传统的高股息蓝筹股
